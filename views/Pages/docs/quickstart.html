Quickstart

Kucluck Bot adalah bot Discord serba guna yang gratis dan berkualitas untuk server Discord kalian. Kucluck bot sangat mudah di invite ke server Discord kalian tanpa konfigurasi yang rumit.

Invite Bot

Mengundang Kucluck Bot di server kalian sangat mudah. Pertama, silahkan periksa apakah Kamu memiliki izin Administrator atau Kelola Server.


Untuk pemilik server memiliki semua izin bahkan tanpa peran.

Jika kalian sudah dipastikan memiliki peran dalam mengelolah server, sekarang Kamu bisa invite Kucluck Bot dengan menggunakan link ini

Menggunakan Kucluck Bot

Sekarang Kamu sudah bisa menggunakan Kucluck Bot di server Discord Kamu. Kamu bisa lihat commands untuk Kucluck Bot disini, atau kamu bisa mengirimkan pesan !help [command] yang nantinya Kucluck Bot akan mengirimkan DM ke kamu tentang semua commands Kucluck Bot.
